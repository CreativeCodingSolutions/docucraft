<li class="ml-6 list-disc text-muted-foreground"><strong>New contributors</strong> struggle to understand the context of changes</li>
<li class="ml-6 list-disc text-muted-foreground"><strong>Changelogs</strong> are harder to generate without structured descriptions</li>
<li class="ml-6 list-disc text-muted-foreground"><strong>Bus factors</strong> increase when only the author understands the change</li>

<h2 class="mt-10 mb-4 text-2xl font-bold tracking-tight">How DocuCraft Fixes This</h2>
<!-- -->
<p class="leading-relaxed text-muted-foreground">DocuCraft is a GitHub Action that runs on every PR and generates a structured description automatically. It analyzes the diff, classifies changes, and generates a consistent, readable description.</p>
<!-- -->
<p class="leading-relaxed text-muted-foreground">It works on every PR — no API keys, no configuration, just add the workflow.</p>
